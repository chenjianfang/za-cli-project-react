## Introduction
@za-cli/project-react是用来生成react项目的构建配置

## Installation
```
npm install @za-cli/project-react -g -f
```

## Usage
```
// 生成react项目
za-init-react

// 进入创建的文件夹
cd yourProjectName

// 开发react项目
npx za-dev-react

// 打包生产react
npx za-prod-react

// 输出配置，以后打包用输出的配置
npx za-eject-react

```
